Flexbox: alinha os elementos na horizontal, é dividido em duas partes:

  1. FlexContainer: Div que contém os Flex-itens, sendo o elemento pai de outros elementos
    • Flex-direction: muda a orientação do eixo primário dos elementos na vertical ou na horizontal
      1. row - default: alinha os elementos horizontalmente
      2. row-reverse: alinha os elementos horizontalmente de forma inversa
      3. column: alinha os elementos verticalmente
      4. column-reverse: alinha os elementos verticalmente de forma inversa
    • Flex-wrap
      1. nowrap : deixa sem quebra de linha
      2. wrap: adiciona quebra de linha com o redimensionamnto da janela. Por padrão, o flexbox não quebra a linha
      3. reverse: reverte os elementos com quebra e linha
    • Flex-flow: atalho para executar o flex-direction e o flex-wrap em um comando
      1. row nowrap
      2. row wrap
      3. column wrap
    • Justify-Content: forma de alinhar os elementos no container, de acordo com o eixo primário
      1. start: alinha no começo do container
      2. flex-end: alinha no final do container
      3. center: alinha no centro do container
      4. space-between: alinha os elementos com espaços entre cada um
      5. space-around: distribuição igual dos elementos
    • Align-items: forma de alinhar os elementos, de acordo com o eixo segundário (inverso do primário)
      1. flex-start: alinha os itens no início
      2. flex-end: alinha os itens no fim
      3. center: alinha os itens no centro
      4. stretch: estica os itens
      5. baseline: alinha os elementos com base no tamanho das letras

      ** gap: adiciona espaço entre os elementos
